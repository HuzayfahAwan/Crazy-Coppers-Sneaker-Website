# Crazy Coppers — Most Expensive Sneakers

A sleek, dark-themed sneaker showcase website featuring the 9 most expensive and sought-after sneakers on the resale market. Originally built in 2022 and redesigned in 2026 with a full visual and feature overhaul.

## What It Does

Crazy Coppers lets you browse ultra-rare, high-value sneakers — from Virgil Abloh's final Louis Vuitton x Nike collab to the holy grail of SB Dunks — with direct links to purchase them on GOAT and StockX.

## Features

- **Dark / Light mode** — toggle between themes, preference saved across sessions
- **Most Wanted spotlight** — full-width hero section for the #1 most expensive sneaker
- **Animated stat counters** — combined collection value, sneaker count, and highest price count up on load
- **Search** — live-filter cards by sneaker name as you type
- **Brand filters** — filter by Louis Vuitton, Dior, Jordan, Nike, Travis Scott, Off-White, and Undefeated
- **Sort by price** — switch between highest-first and lowest-first ordering
- **Card flip** — tap the ⓘ button on any card to flip it and reveal a fun fact about that sneaker
- **Skeleton loaders** — shimmer placeholders while sneaker images load
- **Scroll animations** — cards fade in as you scroll down the page
- **Share links** — copy a direct link to any individual sneaker card
- **Toast notifications** — feedback messages on actions like copying a link or clicking Cop It
- **Fully responsive** — 3-column grid on desktop, 2 on tablet, 1 on mobile

## Tech Stack

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks or dependencies beyond a Google Font. All interactivity (filtering, sorting, animations, theme switching) is handled in a single `script.js` file.

## Running It

Just open `index.html` in any browser. No build step, no server required.
